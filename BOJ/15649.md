# 백준 알고리즘 15649번, ['N과 M'](https://www.acmicpc.net/problem/15649)

`백트래킹` 

## 소스

### kotlin

```kotlin
import java.util.*

var n = 0
var m = 0
val arr = IntArray(10)
val isUsed = BooleanArray(10)

fun main() = with(Scanner(System.`in`)) {
    n = nextInt()
    m = nextInt()
    backTrack(0)
}

fun backTrack(k : Int) {
    if (k == m) {
        for (i in 0 until m) {
            print("${arr[i]} ")
        }
        println()
        return
    }

    for (i in 1 .. n) {
        if (isUsed[i].not()) {
            arr[k] = i
            isUsed[i] = true
            backTrack(k+1)
            isUsed[i] = false
        }
    }
}
```
